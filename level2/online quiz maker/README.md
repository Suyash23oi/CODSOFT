# Online Quiz Maker

A simple responsive quiz platform that allows users to register, create quizzes, browse quizzes, take quizzes, and view immediate results.

## Features

- User registration and login.
- Create quizzes with multiple-choice questions.
- Browse available quizzes.
- Take quizzes one question at a time.
- View final score and answer review.
- Responsive design for mobile and desktop.

## How to use

1. Open `index.html` in a browser.
2. Register a new user or log in with an existing account.
3. Create a quiz from the dashboard.
4. Browse quizzes and take one.
5. Submit answers and review your score.

## Notes

- Data is stored locally in browser `localStorage`.
- No backend required.
